The Dev Challenge {1.0}

For today and tomorrow your goal is to build the technical proof to demonstrate that you are someone that a startup should hire. Individually, or in pairs, your goal is to flesh out some GitHub repositories with code that you can point to in an interview and be proud. You will be demoing what you build on Friday morning.

To help you scope the work, the next page has some examples to inspire you. Your goal is to build the social proof that you have some coding chops.

Go. Build. Make #awesome.

Inspiration

To help you scope the work, below are some examples to inspire you.

Rails
Twitter clone, classified ads system (a better craigslist), to-do list manager, anything that connects with Twitter or Facebook.

jQuery/ frontend
Anything sexy, mashup of APIs like Twitter or Facebook, jQuery mobile interfaces.

Node.js
Real-time visualization (Twitter, Instagram are easy), Twilio integration (SMS/voice), real-time chat, connecting people through the browser.

APIs
Consider using hot APIs like Twitter, Facebook, Twilio, Wolfram Alpha, The Echo Nest, Google Maps API, Foursquare, Netflix, Geocoda, Geoloqi, Factual.
